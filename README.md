# 一个伟大的项目

## 依赖项

安装[nodejs](https://nodejs.org/en)
安装vue3

```npm
npm install vue -g
```

```python
pip install django-cors-headers
pip install django
torch
pip install imageio
pip install numpy
pip install tqdm
pip install opencv-python
```

## 快速开始-运行前端

``` s
cd web
npm run serve
```

部署时使用

```s
cd web
npm run build
```

## 快速开始-运行后端

进入backend目录运行

```s
cd backend
python manage.py runserver 3000
```
